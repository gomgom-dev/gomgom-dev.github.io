# gomgom-dev.github.io

gomgom-dev GitHub Pages 대표 사이트입니다.

## Structure

- `index.html` - 전체 프로젝트 허브
- `projects.js` - 프로젝트별 공통 메타데이터
- `site.js` - 루트 프로젝트 디렉터리 렌더링
- `project-page.js` - 하위 폴더 프로젝트 페이지 렌더링
- `*/index.html` - 프로젝트별 독립 Pages URL

## Account Rule

이 폴더의 git 작성자 설정은 `gomgom-dev` 전용으로 유지합니다.
다른 로컬 폴더의 전역 `jhny-kor` 설정은 변경하지 않습니다.
